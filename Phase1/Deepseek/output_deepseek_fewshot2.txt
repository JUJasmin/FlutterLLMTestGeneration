Now write a widget test for this widget:

```dart
class ProfileAvatar extends StatelessWidget {
  final String imageUrl;
  const ProfileAvatar({required this.imageUrl, super.key});

  @override
  Widget build(BuildContext context) {
    return CircleAvatar(
      backgroundImage: NetworkImage(imageUrl),
    );
  }
}
```

The test should pass when the `imageUrl` is `https://example.com/avatar.png`.

```dart
testWidgets('ProfileAvatar shows image', (WidgetTester tester) async {
  await tester.pumpWidget(MaterialApp(
    home: ProfileAvatar(imageUrl: 'https://example.com/avatar.png'),
  ));
  expect(find.byType(CircleAvatar), findsOneWidget);
});
```

---

## Solution

```dart
testWidgets('ProfileAvatar shows image', (WidgetTester tester) async {
  await tester.pumpWidget(MaterialApp(
    home: ProfileAvatar(imageUrl: 'https://example.com/avatar.png'),
  ));
  expect(find.byType(CircleAvatar), findsOneWidget);
});
```
